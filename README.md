# 《星火》抗日主题游戏

## 游戏介绍

《星火》是一款以抗日战争为主题的文字冒险+生存模拟游戏。玩家将扮演不同身份的角色，体验从1937年七七事变到1945年抗战胜利的全过程，通过多视角叙事展现抗日战争的全貌。

## 游戏特色

- **多视角叙事**：扮演4个不同身份的角色，体验不同战场和领域的抗战
- **历史真实性**：基于真实历史事件改编，展现抗日战争的各个方面
- **深度主题**：探讨人民战争、统一战线、文化抗战、经济抗战等重要主题
- **选择与后果**：每一个选择都会影响剧情发展和结局
- **资源管理**：管理有限资源，在艰苦的环境中生存和战斗
- **情绪体验**：从绝望到胜利的完整情绪曲线
- **成就系统**：8种成就等你来解锁
- **随机事件**：多种随机事件增加游戏乐趣
- **肉鸽支持**：多种支持类型，增强游戏体验
- **完成度统计**：追踪你的游戏进度

## 游戏玩法

1. **选择角色**：从普通村民、国军士兵、文化工作者、商人中选择初始角色
2. **推进剧情**：按照时间线推进剧情，体验不同视角的抗战生活
3. **管理资源**：合理分配食物、弹药、情报、声望、金钱、药品等资源
4. **做出选择**：在每个剧情节点做出选择，影响剧情发展
5. **解锁结局**：根据选择和游戏进程，解锁不同的结局
6. **收集成就**：完成特定条件解锁成就

## 游戏资源

资源包括：
- **食物**：维持生存的基本资源
- **弹药**：进行战斗的必要资源
- **情报**：了解敌情的关键资源
- **声望**：获得民众支持的重要资源
- **金钱**：购买物资的通用资源
- **药品**：治疗伤员的珍贵资源

## 游戏主题

- **人民战争和统一战线**：展示全民抗战的力量
- **敌后战场和正面战场的结合**：通过暗线系统展示两个战场的相互配合
- **文化和经济领域的抗战**：体现抗战的多方面性
- **大东亚共荣的虚伪**：揭示日本侵略者的谎言
- **军国主义对人的异化**：展示战争对人性的摧残

## 运行方式

1. 克隆或下载游戏文件到本地
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏

## 技术实现

- **前端技术**：HTML5 + CSS3 + JavaScript
- **游戏引擎**：自定义简单引擎，无需复杂框架
- **适配**：支持PC和移动设备
- **存储**：使用localStorage保存游戏进度
